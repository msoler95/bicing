Bonus: Ejercicio 1

Enunciado: Para el caso de una estación de bicicletas, en el que como máximo hay 30 bicis. Queremos saber cuantas veces ha habido menos bicis libres que un determinado valor X


Implementación guardarNuevoValor(x)

Como sabemos que como máximo habrá 30 bicis, podemos crear un array de tamaño 31 (la posición 0 no cuenta como una bici) y que cada posición represente un número de bicis introducidas. Inicializaremos cada posición del array a 0. Cada vez introduzcamos un numero de bicis X, se sumará una unidad a la posición X. Ejemplo: '9.00h 5' sumaria una unidad a la posición 5.

Implementación vecesBicisLibresMenorQue(x)

Para saber el numero de bicis libres solo tendriamos que recoger los valores desde la posición 0 hasta la posición x-1. Ejemplo: vecesBicisLibresMenorQue(5) recogeriamos los valores guardados desde las posición 0 hasta la 4 y los sumariamos.
